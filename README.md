🦖 공룡별 위기대처법 🦖

# 1. 실행 방법

- 실행 환경

* 웹 브라우저(Chrome, Microsoft Edge 권장)

- 실행 방법

1. 프로젝트 폴더를 다운로드하거나 GitHub Repository를 Clone합니다.
2. `dinosaurtest.html` 파일을 실행합니다.
3. 웹페이지가 실행되면 질문에 답을 선택하고 결과를 확인합니다.

---

# 2. 작품 소개

'공룡별 위기 대처법'은 사용자의 위기 대처 성향을 공룡의 특징과 연결하여 재미있게 알아볼 수 있는 웹 기반 테스트입니다.

시험을 망쳤을 때, 친구와 갈등이 생겼을 때, 계획이 틀어졌을 때와 같은 다양한 상황에서 사용자의 선택을 분석하여 가장 비슷한 공룡 유형을 결과로 제공합니다.

---

# 3. 주요 기능

* 위기 상황에 대한 3개의 질문 제공
* 4개의 선택지 중 하나를 선택하여 진행
* 선택 결과에 따른 공룡 유형 분석
* 결과 화면에서 공룡 이미지와 설명 제공
* 여러 유형이 동일한 점수인 경우 '초특급 공룡형' 결과 제공
* 다시하기 버튼을 통해 테스트 재시작 가능

---

# 4. 기술적 포인트

* HTML, CSS, JavaScript만 사용하여 서버 개발 없이 실행 가능하도록 구현
* JavaScript를 이용하여 질문 전환과 결과 계산 기능 구현
* 객체(Object)와 배열(Array)을 활용하여 질문 및 결과 데이터를 효율적으로 관리
* 점수 기반 알고리즘을 적용하여 사용자 성향을 분석
* 동점 발생 시 별도의 결과를 제공하여 다양한 선택 패턴도 표현

---

# 5. 느낀 점

단순한 성향 테스트를 넘어 공룡의 생존 전략과 사람의 위기 대처 방식을 연결하는 아이디어를 웹으로 구현해 보았습니다. 1학기 전공 수업에서 못해본 Web 관련 코딩 경험을 가질 수 있어 낯설지만 유익하고 재밌는 시간이었습니다.
